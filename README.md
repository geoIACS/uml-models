# geoIACS UML Model

The purpose of this repository is to manage the changes to the geoIACS UML model.

Only the latest version of UML model is available as an [interactive HTML view](https://geoIACS.github.io/uml-models/html). This view includes detailed definitions of spatial object types, data types and code lists.

Older versions of UML models are available in the corresponding TG. The related html views are not published in this repository and are available only as static html files.
